# COCOJOJOR Services Informatiques — Site vitrine

Site web statique, responsive et single‑page pour **COCOJOJOR SERVICES INFORMATIQUES**, développé avec **Vite**, **Alpine.js** et **Tailwind CSS**. Prêt à être déployé sur **Vercel** (ou tout autre hébergeur statique).

---

## 📸 Aperçu

Le site présente :

- Une **hero section** percutante avec les valeurs de l’entreprise (Rapide, Intelligent, Fiable, Sympa).
- Une grille de **services IT** : installation système, logiciels, maintenance, dépannage, vente de matériels, conseils.
- Une section **“Pourquoi nous”** mettant en avant les atouts différenciants.
- Des **coordonnées complètes** (téléphone, email, site web) avec liens cliquables.
- Un **formulaire d’inscription à la newsletter** (simulation avec feedback visuel).
- Un **footer** avec réseaux sociaux, liens rapides et mentions légales.
- Un **bouton “retour en haut”** et une **navigation responsive** (menu mobile coulissant).

---

## 🧱 Technologies utilisées

| Technologie       | Rôle                                      |
|-------------------|-------------------------------------------|
| **Vite**          | Bundler et serveur de développement ultra‑rapide |
| **Alpine.js**     | Interactions légères (menu mobile, formulaire, retour en haut) |
| **Tailwind CSS**  | Framework CSS utilitaire pour un design personnalisé et responsive |
| **Font Awesome 6**| Icônes vectorielles                      |
| **Google Fonts** (Inter) | Typographie moderne                 |
